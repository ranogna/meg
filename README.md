# Happy Birthday Mobile App

This is a simple mobile-friendly web page that can open directly on a phone.

## How to use

1. Copy the `index.html` file to your phone or send it by WhatsApp.
2. Open `index.html` in the phone browser.
3. The page will open directly and show the birthday message.

## If you want a website link

You can host this page online with any static site host:

- GitHub Pages
- Netlify (drag and drop the folder)
- Vercel

That will give you a URL your brother can open directly from WhatsApp.

## Note

This version does not ask the birthday person to add a photo. It opens with a ready-made mobile greeting card.
